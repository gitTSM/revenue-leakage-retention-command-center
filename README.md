# Revenue Leakage & Retention Command Center

## Overview
This project simulates a real-world enterprise BI solution designed to identify revenue leakage and improve customer retention.

## Business Problem
Organizations often lack visibility into where revenue is lost across renewals, billing, and customer behavior.

## Solution
An end-to-end analytics solution using PostgreSQL and Tableau to:
- Identify revenue leakage drivers
- Analyze customer retention risk
- Provide executive-level insights
- Enable data-driven decision making

## Tools & Technologies
- PostgreSQL (pgAdmin)
- SQL
- Tableau
- GitHub

## Project Structure
- `/docs` → business documentation
- `/sql` → data modeling and transformations
- `/tableau` → dashboard files
- `/images` → visuals and previews
